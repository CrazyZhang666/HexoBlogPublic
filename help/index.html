前言

在解决问题之前，玩家先检查下以下步骤是否完成

  1. 小助手版本是最新版本
  2. 运行库版本是最新版本
  3. 已退出或关闭杀毒软件

打开没反应

这个一般分为两种情况，一种是Windows系统会自动阻止从互联网下载的exe文件，一种是运行库版本过低

属性中解除锁定

如果小助手打开没反应，请在小助手文件上点击右键->属性->勾选解除锁定->点击应用，然后再次打开小助手即可

更新运行库到最新版本

过低的运行库版本也可能导致小助手出现问题，大家可以更新下NET版本到小助手推荐版本（不需要卸载旧版本，直接覆盖安装就可以更新）

安装最新版本运行库
https://crazyzhang.cn/runtime/

Windows 已保护你的电脑

如果提示这个，我们点击左上角更多信息

然后选择仍要运行即可

软件崩溃或闪退

小助手发生崩溃后，会在这个路径下生成崩溃日志

文档 \ GTA5OnlineTools \ Logger \ Crash

你可以把这些崩溃日志从以下任意方式发送给开发者，以帮助开发者快速定位问题

在没有错误日志的情况下诊断任何问题无异于闭眼开车

1. 重置配置文件

清空小助手配置文件夹里的全部文件，然后再打开小助手

如果你有重要配置文件（比如：kiddion脚本、自定义传送、自定义载具），请先备份，以避免配置文件丢失

文档 \ GTA5OnlineTools

2. 不要放到桌面运行

有的玩家喜欢把小助手放到桌面运行，一般情况下桌面都是位于系统C盘，这可能会导致一些问题。

建议大家在其他盘非C盘系统盘，比如D盘建一个文件夹，然后把小助手单独放到此文件下运行

桌面可以创建小助手快捷方式

3. 右键管理员运行

如果权限不足，可能导致小助手崩溃。如果出现了崩溃问题，大家可以尝试在启动小助手前右键鼠标选择以管理员身份运行

4. 关闭杀毒软件

杀毒有时候会直接删除小助手文件或者会拦截小助手功能导致功能无效或者程序崩溃

这个只需要在使用小助手的时候退出或关闭杀毒软件，或者将小助手文件加入杀毒白名单

Kiddion 由于加了壳，容易被杀毒标记为病毒，Xenos64注入器 已被杀毒标记为病毒（误报，就像激活软件也会被杀毒报毒那样）

关闭Windows安全中心教程，也就是win10/win10系统自带杀毒
https://crazyzhang.cn/gta5/defender.html

如果是第三方杀毒软件，请自行搜索关闭办法

补充说明

比如下面崩溃日志显示的错误信息就是杀毒拦截导致软件没有权限最终引发崩溃（这个拦截过程是没有提示的）

仅完成部分的 ReadProcessMemory 或 WriteProcessMemory 请求

其他情况

如果以上办法都还解决不了你的问题，请到在网站留言或者到Discord里联系我帮忙解决
