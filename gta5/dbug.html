如何正确反馈小助手Bug

更新于2023-02-26|gta5
|字数总计:574|阅读时长:1分钟|阅读量:

前言

看到有些玩家反馈小助手Bug描述的十分笼统，站长也只能靠猜测问题，这样一个问题往往很难得到解决，所以站长写一篇如何正确反馈小助手Bug教程供大家参考

准备

一些常见问题，网站上已经给出解决方案，请先阅读再在考虑继续提问

https://crazyzhang.cn/help/

小助手崩溃

如果小助手程序发生崩溃（注意：不是GTA5游戏崩溃），程序会生成一个崩溃日志

位置：我的文档\GTA5OnlineTools\Log

这个 Crash 文件夹里就是崩溃日志

崩溃日志的文件夹都是按照时间命名的，请查看最新日期的崩溃日志

请将最新的崩溃日志上传到 Github Issues

https://github.com/CrazyZhang666/GTA5OnlineTools/issues

就像这样发起一个新 Issues 即可，如果没有日志对于开发者来说无异于闭眼开车

如果 GitHub 无法访问，可以将日志粘贴到小助手留言板

第三方

这里的辅助基本都是来源于国外 UnKnoWnCheaTs 论坛，站长也只是汉化，程序本身一般是没啥问题的

如果出现问题，站长大概率也是束手无策，站长只能保证汉化没问题

Kiddion一般报错都是旧版本脚本导致的，删除脚本再打开Kiddion即可

功能

这里的辅助是小助手自己设计的，注意，每个功能入口都是相互独立的
